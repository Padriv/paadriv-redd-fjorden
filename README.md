# Paadriv-redd-fjorden
Nettside for Pådriv for "Oppdrag: Fjorden vår", en åpen plattform der aktører kan melde seg som bidragsytere, synliggjøre sitt arbeid og koordinere innsats for fjorden.

## Oppdrag: Fjorden vår - Nettside

Landingsside og bidragsportal for Oppdrag: fjorden vår, et samarbeid mellom nesten 30 aktører som jobber for fjordens fremtid.

## Formål
Nettsiden er en nøytral møteplass der organisasjoner, bedrifter og enkeltpersoner kan:

- Melde seg som bidragsytere til prosjektet
- Synliggjøre sitt bidrag og sin rolle
- Holde seg oppdatert om hva som skjer
- Koordinere innsats med andre aktører

## 🐶 API-klient (Bruno)

Prosjektet inkluderer en [Bruno](https://docs.usebruno.com/get-started/bruno-basics/download)-samling under `bruno-collections/` for å teste Airtable-endepunktene.

**Installer Bruno:**
```bash
brew install bruno
```

Åpne `bruno-collections/`-mappen som en samling i Bruno. Legg til et gyldig Airtable Personal Access Token (PAT) i miljøvariabelen `PAT` i `TEST`-miljøet.
